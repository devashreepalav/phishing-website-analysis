# Phishing Website Analysis

Data science project to analyze phishing website characteristics and predict phishing risk.

- Exploratory Data Analysis (EDA)
- Hypothesis testing (Chi-square & T-test)
- Logistic Regression with class_weight="balanced"
- Random Forest comparison
- ROC curve, confusion matrix, feature importance, and risk scoring (Low / Medium / High)

Main file: `phishing_website_analysis.ipynb`
